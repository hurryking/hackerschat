About


What is Hackers Chat?

Hackers Chat is a user driven community site built around the idea of topics (like subreddits), where each topic has a forum and a chat room. Topics are currently focused on cryptocurrencies, programming and technology, but will expand as the userbase grows. You can either join existing communities that you find interesting, or you can create your own community.

Note: The site is still under development.

The source code for this site is available on Github.

Announcements about Hackers Chat will be made in one or more of the following places :

  • My blog
  • My twitter
  • The Hackers Chat subreddit
  • The Hackers Chat topic on this site
